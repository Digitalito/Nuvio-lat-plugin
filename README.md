# Nuvio Pero Latino

Fork de varios repositorios de plugins latinos con las URL corregidas y con algunas mejoras.

## Como Instalar?
1. Abre **Nuvio** > **Ajustes** > **Plugins**
2. Añade esta URL del repositorio:
```
https://raw.githubusercontent.com/Digitalito/Nuvio-lat-plugin/refs/heads/main/manifest.json
```
3. Actualiza y habilita los proveedores que desees.
